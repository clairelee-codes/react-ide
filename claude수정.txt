■ 8. 모달 내부에서 컨텍스트 함수를 그대로 감싸기만 하는 지역 함수
--------------------------------------------------------------------
- src/Providers/Modals/CreatePlaygroundModal.jsx:10-12
- src/Providers/Modals/CreateFolderModal.jsx:10-12

  const closeModal = () => { modalFeatrues.closeModal(); };

  ModalContext에서 이미 closeModal을 받아올 수 있는데, 굳이 같은 이름의
  지역 함수로 한 번 더 감싸고 있어 "이 closeModal이 컨텍스트 함수인지
  지역 함수인지" 헷갈릴 수 있습니다.

→ 추천: 다른 모달들(CreateCardModal, UpdateFileTitleModal 등)처럼
  const { closeModal } = useContext(ModalContext); 로 바로 구조분해해서
  사용하고, 굳이 감싸야 한다면 이름을 handleClose 등으로 구분해서 사용
